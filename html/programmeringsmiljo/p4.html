Nettbaserte utviklingsmiljø

I tillegg til teksteditorene vi nevnte tidligere, finnes det også steder på nettet hvor du kan skrive programkoden og få ut resultater. Eksempler på disse er

  • codepen.io
  • jsfiddle.net
  • jsbin.com

En ting som er felles for disse er at i tillegg til JavaScript så kan vi også skrive HTML og CSS. Disse to henger nært sammen med JavaScript. En nettside er vanligvis skrevet med HTML kode, og som regel er det inni denne koden igjen vi kjører JavaScript. Når vi programmerer med en teksteditor så lager vi gjerne en html-fil først (selve nettsiden). Deretter skriver vi enten JavaScript rett inn i den filen, eller vi lager en egen JavaScript-fil som vi kobler sammen med html-filen.

På de sidene som nevnt ovenfor trenger vi ikke nødvendigvis å koble sammen en html-fil med JavaScript-filen. Det eneste vi trenger er et vindu for å skrive JavaScript samt et konsoll-vindu for å se ting vi ønsker å skrive ut.
