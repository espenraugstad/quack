Hvor kjører vi koden

Når vi har skrevet inn den koden vi ønsker, så må vi kjøre koden en eller annen plass. Noen programmeringsspråk krever at vi først kompilerer koden. Det vil si vi oversetter den til instruksjoner som en datamaskin kan forstå og kjøre. En kompilator1 er spesifikk for hvert programmeringsspråk og hver maskintype. Eksempler på programmeringsspråk hvor koden må kompileres før den kan kjøres er

  • Java
  • C
  • C++
  • C#

Andre programmeringsspråk trenger ikke kompileres, men tolkes direkte av et annet program på datamaskinen. Eksempler på slike fortolkede programmeringsspråk er

  • Python
  • PHP
  • JavaScript

Et programmeringsspråk som JavaScript kjøres i programmer som så og si alle datamaskiner har i dag, nemlig nettleseren. Hvis du har en nettleser så har du altså en plass å kjøre et program skrevet JavaScript. Det finnes også muligheter til å kjøre JavaScript programmer uten en nettleser, men det kommer vi ikke inn på i dette kurset.
